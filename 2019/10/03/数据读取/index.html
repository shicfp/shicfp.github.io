数据读取

数据获得

一般而言，数据分析的第一步是需要获取数据，不同的数据拥有不同的来源，常见的有以下四种：

  1. 来自数据接口的官网数据：

    一般通过API请求获取，比较典型的例如天气网站的数据等；

  2. 数据库的数据

    需要对方授权才可以访问使用

  3. 使用数据爬虫获取

    这是当下python最为火热的原因，但由于诸多反爬机制的存在，事实上也存在一些限制。常用的有两种方式：

    • 自己编写代码

      • Requests: 抓取网页信息
      • XPath: 通过XML路径语言解析信息
      • Pandas：保存数据并写入XLS/MySQl中

    • 使用工具软件

      • 火车采集器
      • 八爪鱼（支持付费云采集）
      • 集搜客（可视化效果强）

      参考教程： 实验楼爬虫教程

  4. 其他（日志采集等）

数据读取

常用文件的读取方式如下：

csv文件

用一段代码来表示如下：

with open('xxx.csv', r) as f: 
    ....

xlsx文件

通常使用xlrd包来进行解析

file = xlrd.open_workbook(file)
table = file.sheet)by_index(0)

备注：以上文件最为推荐的打开方式为通过pandas来读取，好处有两点：

  1. 代码简洁
  2. 读取出来的数据即为DataFrame格式，方便进行数据清洗等后续工作

pandas支持的读取文件类型

pd.read_csv
pd.read_html
pd.read_clipboard # 剪切板数据
pd.read_excel
pd.read_hdf
pd.read_feather
pd.read_msgpack
pd.read_stata
pd.read_sas
pd.read_pickle

使用时注意每个语句都有不同的参数。

参考文档：官方文档

数据库文件

数据库文件的读取需要安装第三方库，例如读取MySQL文件时，需要安装MySQLdb

参考文档：实验楼教程
